# Reports

This folder collects past audit reports produced by `changelog-release-notes-guardian` when invoked in audit mode.

Audit reports are placed here at: `reports/<YYYY-MM-DD>-audit-<product-name>.md`

The report template lives at `templates/audit-report.md`. The worked example lives at `examples/audit-report-example.md`.

---

*This folder is initially empty. Reports accumulate over time as audits are run.*
