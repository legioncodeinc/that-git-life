# What is That Git Life?

**That Git Life** (TGL) is a one-shot installer and always-on local service from [The Notorious Llama](https://www.thenotoriousllama.com) that sets up your AI-coding environment and keeps your GitHub folder healthy.

## What you get

- **One command** installs everything you need to start vibe coding: GitHub CLI, Node, Python, your AI-native IDE (Cursor *or* VS Code + Claude Code), Claude Desktop, Obsidian, and the TGL package itself.
- **A web dashboard** at `http://localhost:3050` that lives in your browser and shows the health of every repo in your GitHub folder.
- **A standardizer** that gives every new repo (and migrates every existing one to) the same canonical folder structure.
- **A scanner** that catches uncommitted changes, stale branches, missing files, and secrets you didn't mean to commit.
- **A skills feed** that keeps your IDE stocked with up-to-date global skills from the Notorious Llama community.

## What you don't have to do

- Manually install seven tools and read seven READMEs.
- Hunt for the "right" folder structure on every new repo.
- Notice three months later that you've been sitting on uncommitted changes in twelve different repos.
- Copy and paste an SSH key into GitHub's settings page.

## Who it's for

Solo devs, indie hackers, and small teams who lean into AI tooling and want their environment to feel intentional. If "vibe coding" is in your vocabulary, TGL is for you.
