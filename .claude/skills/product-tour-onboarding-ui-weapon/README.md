# product-tour-onboarding-ui-weapon

Procedural arsenal for `product-tour-onboarding-ui-guardian`. This weapon encodes the research-backed decision framework, integration playbooks, and maintenance protocols for the in-app product tour and onboarding UI domain.

It covers platform selection (Userpilot, Appcues, Userflow, Pendo Guides, Driver.js, Shepherd.js, Intro.js), component implementation (tooltips, modals, hotspots, checklists), segment-based trigger logic, and the maintenance protocol that prevents tours from breaking when UI changes.

See the Command Brief at `ai-tools/command-briefs/product-tour-onboarding-ui-guardian-command-brief.md` and the research summary at `research/research-summary.md` for context on how this weapon was forged.
