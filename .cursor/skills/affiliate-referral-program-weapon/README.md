# Affiliate and Referral Program Weapon

Practitioner arsenal for `affiliate-referral-program-guardian`. This skill encodes everything needed to select, configure, instrument, economically validate, and protect a SaaS affiliate or referral program in 2026.

## Purpose

SaaS founders launching affiliate or referral programs face three failure modes: wrong platform choice, broken attribution (especially for Safari users), and fraud that drains commissions before ROI is proven. This weapon provides the decision matrices, architecture playbooks, and fraud checklists to avoid all three.
