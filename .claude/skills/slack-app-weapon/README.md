# slack-app-weapon

The Slack app development playbook for `slack-app-guardian`. Covers the full Slack developer surface — Bolt SDK setup (JS/Python/Java), slash commands, Block Kit UI, modals, Events API, OAuth multi-workspace install, and App Directory/Marketplace submission — with research-backed, opinionated guidance on the top production failure modes.

See the [Command Brief](../../command-briefs/slack-app-guardian-command-brief.md) for the full scope definition and the [research summary](research/research-summary.md) for the five most influential sources and open questions that informed this weapon.
