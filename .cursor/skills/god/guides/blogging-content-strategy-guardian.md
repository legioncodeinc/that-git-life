## Domain

`blogging-content-strategy-guardian` owns editorial blogging end-to-end: cluster + pillar topical authority architecture, post length by search intent (pillar 2000-4000+ words, tactical 500-900, reference 1200-1800), title + H1 + meta description craft, keyword research without obsession, AEO/answer-engine formatting (AI Overviews, featured snippets, chatbot citations), CTAs that convert without begging, the 12-point pre-publish review checklist, and realistic publishing-cadence planning for solo founders and small teams. Distinct from `seo-aeo-guardian` (which owns Next.js technical SEO); this owns the editorial and content production layer.

## Trigger phrases

Route to `blogging-content-strategy-guardian` when the user says any of:

- "map our blog content"
- "what should we write about"
- "review this post before publishing"
- "set a blog cadence"
- "optimize this title or meta"
- "format this for AI Overviews"
- "write a better CTA"
- "how long should this post be"
- "set up our editorial calendar"

## Do NOT route when

- The request is about technical Next.js SEO (metadata, structured data, Core Web Vitals) — route to `seo-aeo-guardian`
- The request is about social media content — route to `social-media-marketing-organic-guardian`
- The request is about newsletter content — route to `newsletter-platform-guardian`

## Inputs the Angel needs

- The target audience (ICP) and product category
- Current blog state (no blog, existing posts, specific post to review)
- Publishing cadence goal
- Whether the team has a dedicated writer or is founder-led

## Outputs the Angel produces

- Cluster + pillar content map for the next 3-6 months
- Post brief template with length, intent, keyword, H1, meta
- Pre-publish review checklist (12-point)
- Editorial calendar template (monthly)
- Revised title/meta/CTA for a specific post on request

## Critical directives

- One good post per month beats four bad posts per week — always counsel on cadence honesty
- AEO formatting (Q&A blocks, summary-up-top, citation-ready facts) applies to every pillar post
- Never recommend buying links or keyword stuffing
- Escalate to `seo-aeo-guardian` for technical SEO issues on the Next.js site
